# 테스트 보정 체크리스트

상태: draft
기준일: 2026-04-24

## 목적

실제 구현과 통합 검증 과정에서 테스트 후 확정해야 하는 항목을 한 곳에서 관리한다.

## 현재 상태

- 구조 설계는 정리되었고, 일부 수치와 세부 정책은 테스트 후 확정 예정이다.
- 이 문서는 M7 통합 검증 시 보정해야 하는 항목의 체크리스트다.

## 범위

- 데이터 플레인 보정
- Risk 모델 보정
- 입력/시나리오 보정
- 배포/운영 보정

## 상세 내용

## 데이터 플레인 보정

- [ ] 표준 입력 스키마 내부 구조 최종 확정

## Risk 모델 보정

- [ ] 온도 기준값 확정
- [ ] 습도 기준값 확정
- [ ] 초기 가중치 보정
- [ ] Top 3 원인 코드 사전 최종 보정

## 입력 / 시나리오 보정

- [ ] `factory-b` Dummy `normal` 값 범위 확정
- [ ] `factory-b` Dummy `warning` 값 범위 확정
- [ ] `factory-b` Dummy `danger` 값 범위 확정
- [ ] `factory-c` Dummy 시나리오 범위 재검증
- [ ] `null` 허용 필드 및 처리 정책 확정

## 배포 / 운영 보정

- [ ] 배포 지연 시간 실측값 기록

- [ ] Failover 실측값 기록
- [ ] 문서와 실제 경로 불일치 여부 재점검
